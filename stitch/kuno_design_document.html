# KUNO — Design System & UI Specs ## App Identity **Name:** Kuno **Tagline:** Your shelf, your story. **Personality:** Warm, cozy, premium, intimate. Like a personal library in golden hour light. ## Color Palette ### Light Mode - **Background:** `#FAF7F2` (Warm Linen) - **Surface:** `#F2EDE4` (Warm Card) - **Border:** `#E0D5C5` (Soft Border) - **Primary Text:** `#2C2416` (Deep Warm Brown) - **Muted Text:** `#8B7355` (Warm Tan) - **Accent:** `#C17F4A` (Warm Amber-Orange) - **Accent Soft:** `#F0DFC8` (Pale Peach) - **Success (Green):** `#6B8F71` - **Error/Rose:** `#C17179` - **Info (Blue):** `#6B85A0` ### Dark Mode - **Background:** `#1C1A17` - **Surface:** `#252219` - **Border:** `#3A3527` - **Text:** `#F0EAE0` - **Muted:** `#9A8E78` ## Typography - **Headings:** Playfair Display (Serif) - Elegant, warm. - **Body / UI:** Inter (Sans-Serif) - Clean, readable. ## UI Elements - **Radius:** Cards (12px), Chips (8px), Buttons (24px), Pills (999px). - **Shadows:** `0 4px 16px rgba(44, 36, 22, 0.08)` - **Shelf Plank:** Gradient from `#A0784A` to `#8B6335`, 12px height. - **Progress Bar:** Track `#E0D5C5`, Fill (Amber Gradient).
